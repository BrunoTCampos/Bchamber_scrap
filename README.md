# Scraping Câmara dos Deputados

Sobre
-------
  * Script via Phyton para extração das transcrições dos pronunciamentos, por ano, em plenário da Câmara dos Deputados

Instalação das dependências
-------

```bash
$ pip install pip bs4 pandas
```
